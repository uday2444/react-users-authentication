# React Authentication

React Authentication is a web application that provides user authentication functionality. It allows users to register, login, and access protected routes based on their authentication status.

## Project Overview

React Authentication is built using React.js and provides a simple and secure way to handle user authentication in a web application. It uses React Router for navigation and Axios for making API requests to the server.

The main features of React Authentication include:
- User registration
- User login
- Protected routes that can only be accessed by authenticated users
- Session management using session storage

Install the dependencies:

npm install

Install JSON Server globally:

npm install -g json-server

Start the JSON Server to run the fake API server:

json-server --watch db.json --port 3001
